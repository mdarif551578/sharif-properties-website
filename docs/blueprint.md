# **App Name**: Sharif Properties Service

## Core Features:

- Property Listings Display: Display property listings in a grid format with key details such as price, address, and a featured image.
- Client-Side Filtering: Allow users to filter properties based on location, price range, bedrooms, and property type using client-side logic for static export compatibility.
- Property Detail Pages: Provide detailed information for each property, including an image gallery, description, features, and agent contact details.
- Hero Section with Search: Implement a full-screen hero section on the home page with a search bar allowing users to search by location, price, and property type.
- Featured Properties Section: Showcase a selection of featured properties on the home page to attract user interest.
- Contact Form Tool: Implement a contact form where users can submit inquiries. The tool analyzes the form data to classify it (e.g. urgent requests get flagged).

## Style Guidelines:

- Primary color: Navy Blue (#1A2B4C) for a luxurious and professional feel.
- Secondary color: Gold (#D4AF37) as an accent to highlight important elements and CTAs.
- Background color: Off-white (#F8F8FF) to provide a clean and modern backdrop.
- Body and headline font: 'Inter' sans-serif, for a contemporary and readable design. Note: currently only Google Fonts are supported.
- Use modern and minimalist icons to represent property features and amenities.
- Implement a mobile-first, responsive layout using Tailwind CSS grid and flexbox utilities.
- Subtle hover animations on property cards and buttons to enhance user interaction.
